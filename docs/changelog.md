# RunWith Changelog

## v4.1 (2026-03-27) — 코치 기반 러닝 경험

### Added
- **홈**: 오늘의 코치 가이드 카드 (Gold 좌측 보더 스타일)
- **홈**: 지난 러닝 코칭 요약 (Green 카드)
- **워밍업**: GPS / 그룹 매칭 / 코치 데이터 체크리스트 (순차 완료 애니메이션)
- **결과 공유**: 코치 한줄 피드백 + 순위 상승 표시
- **결과 분석**: 코치 해석 카드 (텍스트 분석)
- **결과 분석**: 다음 액션 제안 카드 (Gold 보더)
- **달력**: 날짜 상세 바텀시트에 코치 피드백 섹션

### Changed
- **워밍업**: "124명 참가 확정" → "비슷한 레벨 러너 124명 매칭 완료"
- **러닝 탭**: "러닝 데이터" → "실시간 데이터"
- **러닝 탭**: "페이서 비교" → "코치 리듬 비교"
- **코칭 배너**: 단일 문장 → 2줄 구조 (현재 상태 + 행동 제안)
- **순위 표시**: 강조도 하향 (`opacity: 0.7`), 텍스트 "현재 그룹 위치"로 변경
- **페이서 갭 레이블**: "가이드보다 뒤처짐" → "현재 구간 목표 리듬보다 느림"
- **페이서 코칭**: 거리 중심 → 리듬/구간 중심 문구
- **결과 분석**: "내 기록 라인" → "내 리듬 라인" 범례 텍스트

### Architecture
- `coaching.js` 신규 생성 — 코칭 메시지 중앙 관리 시스템
- `router.js` 분리 — URL 해시 기반 화면 전환
- `timer.js` 분리 — 워밍업/러닝 타이머 + 체크리스트 로직
- `pacer.js` 분리 — 페이서/리듬 비교 로직
- `calendar.js` 분리 — 달력 렌더링 + 바텀시트
